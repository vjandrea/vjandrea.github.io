Preset Pool "Dynamic"

The preset pool Dynamic is a pool that automatically displays the preset type which is currently enabled in the preset control bar. 

Open the Preset Pool "Dynamic"

To open the preset pool Dynamic:

  1. Tap anywhere in the user-defined area.
  2. The pop-up Create Basic Window opens.
  3. Tap Presets. 
  4. Tap Dynamic. 
  5. The preset pool Dynamic opens. 

Close the Preset Pool "Dynamic"

To close the preset pool, tap  in the upper left corner of the pool and tap Delete Window.
