Effect keyword

To enter the keyword into the console, press Effect or type Effect or the shortcut Ef into the command line. 

Description

The keyword Effect is an object keyword. 

Syntax

[Function] Effect [ID]/[Name]/[Number]

 

Example

[Channel]> Move Effect 1 At 37 

Moves the effect 1 to the cell 37 of the effect pool. 

 

[Channel]> SelFix Effect 1

 Selects fixtures in the fixture sheet that can run effect 1 if it is a selective effect.
