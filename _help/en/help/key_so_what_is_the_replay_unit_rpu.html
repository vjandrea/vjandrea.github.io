What is the Replay Unit (RPU)

A grandMA2 replay unit (RPU) is considered as a console in the system.

It has the same software and capabilities as the consoles. It is a 19" 3 Unit rack mount version of the console, without all the user interface and physical controls. It got the same connectors on the back panel as a console (it got less USB connectors).

It functions exactly like a console in the system.
