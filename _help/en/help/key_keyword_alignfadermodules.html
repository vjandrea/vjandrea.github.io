AlignFaderModules keyword

AlignFaderModules is a function used to configure connected external wings.

To execute this keyword type AlignFaderModules or the shortcut AlignF into the command line. 

 

Note:
Wings are fader extensions and are intended to follow the faders built-in in the console.

 

Syntax:

AlignFaderModules

 

Entering AlignFaderModules into the command line automatically sets up a surface with connected faders next to each other without overlapping.

If a default surface already exists, this command sets automatically a second surface. 

 

For more information on how to set surfaces see the keyword Surface.
