MAtricks Wings

This topic illustrates how to use the wings in the MAtricks with the help of an example. 

Wings split the selection into parts. Each wing mirrors the opposite wing. For explanation see 2. Example.

For information on the keyword see MAtricksWings.

Requirement:

  • Fixtures are selected via the command line or keys. 

For more information on how to select fixtures for the usage of MAtricks see MAtricks. 

1. Example

  • Apply wings
Hint:
To move through selection, you usually press Next to jump to the adjacent fixture see illustration below this box.   

Initial situation:

  1. To apply wings, tap the button on the left of the  sign displayed below Wing in the MAtricks toolbar.

For more information see MAtricks toolbar. 

  1. The calculator opens.
    -Enter 2.
  2. Press Please. 

-or-

  1. Use the command line as follows:
[Channel]> MAtricksWings 2
  1. Press Please.
  2. Press Next.  

Result:

The fixtures form two wings from left and right. 

  | 

  1. To move the fixtures toward each other, tap Next.

 2. Example

  1. Set 3 wings using the command line:
[Channel]> MAtricksWings 3
  1. Press Please.
  2. Press Next.

Result: 

The wing on the right – the third wing – mirrors the adjacent wing in the middle.
