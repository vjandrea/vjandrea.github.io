Locate Keyword

To get the Locate keyword in the command line, type Locate or the shortcut Loca in the command line.

Description

The locate keyword is a function keyword to locate assigned objects on executors. The page will change to that page where the object is assigned and a HERE is displayed.

To turn locate off, repeat the Locate keyword with no object.

Syntax

Locate [Object]

Locate

Example

  • To locate sequence 1 and see where it is assigned, type in the command line.
[Channel]> Locate Sequence 1
