Speed Masters

It is possible to a assign 15 different Speed Masters to executors.

This group of masters also include the BMP Master. This master is controlled by the incoming sound and will adjust to the detected Beats Per Minute. Read more about it in the Sound Input topic.

The Speed Masters can be assigned to Sequence executors, chasers or effects. This makes it possible to sync multiple elements to the same speed.

Read more about assigning masters to executors in the Executor Options topic. Read about assigning a Speed Master to an effect in the Assign Effect to Executor topic. 

 

All the Speed Masters can be seen and adjusted in the Masters window.
