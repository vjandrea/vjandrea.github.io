Edit a macro

Editing or changing an existing macro is just like creating the macros. You need to open the Edit Macro pop-up.

You can open this by pressing the Edit key and then tapping the the macro you want to edit.

This will open the Edit Macro pop-up. It could look like this:

Please have a look at the Manually create a macro topic for a detailed explanation about this pop-up.

 

In this editor you can select the line you want to edit and then press the Please/Enter key on the keyboard to open the Enter CMD pop-up.

If you are on a console it might be easier to tap the screen encoder to edit the line.

If you are using an onPC then you can right click the line to edit it.

 

Type the new command. finish by pressing Please/Enter and close the pop-up.

You have now edited the macro. If you want learn more about creating macros, then please read the Manually create a macro topic.
