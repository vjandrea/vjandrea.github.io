Edit keyword

To enter the keyword Edit into the command line, press the key Edit or type Edit or the shortcut E into the command line. 

Description

Edit is a function keyword and it is used to modify values. 

Syntax

Edit [Object]

 

Options

To get a list of all options of the keyword Edit, type the following into the command line:

[Channel]> Edit /?

The keyword Edit has the following options:

Option Shortcut Option Value Description
noconfirm nc No option value Suppresses confirmation pop-up. 

 

Examples

[Channel]> Edit Effect 2

Opens the effect editor to modify the effect 2. 

 

[Channel]> Edit

Takes the first cue in the sequence of a selected executor. If the executor is in a certain cue, then this cue is edited.
