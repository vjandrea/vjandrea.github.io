Freeze Keyword

Important:
If the Freeze keyword is entered in the command line by pressing the Freeze key, the keyword will be directly executed.

To execute the keyword Freeze directly press Freeze.

 

Another way is to type Freeze or the shortcut Fre directly into the command line.

Description

Important:
If Freeze is on, programmer values have a higher priority as playbacks. 
Only executor priority "Super" has a higher priority than Freeze. 

Freeze is function keyword to change the priority of the programmer.

Freeze is a toggle function. This means that entering Freeze without any helping keyword toggles the Freeze-mode on/off.

 

Syntax

Freeze

Freeze On

Freeze Off

 

Example

 

[Channel]> Freeze On

 

Turns the freeze mode on.
