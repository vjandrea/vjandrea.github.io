Mask Keyword

To execute the keyword Mask press and hold 

and Group Group Group or type Mask into the command line.

Description

The Mask keyword is an object keyword to access Masks in the Mask pool. The default function is Select.

For more information, see worlds, filters, and masks.

Syntax

[Function] Mask [ID]

Examples:

 

[Channel]> Mask 3

 

Selects mask 3 in the mask pool.

 

[Channel]> Mask "My Mask"

 

Selects mask with the name "My Mask".
