StepFade Keyword

To get the StepFade keyword, type StepFade or the shortcut Ste in the command line.

Description

With the StepFade keyword, you assign executors as a step fader. A step fader controls the fade time between the steps of a chaser.

For more information, see executors - assign a function.

The StepFade keyword is a function keyword.

Syntax

StepFade [Executor-list]

Examples

  • Assign executor 28 as a step fader.
[Channel]> Assign StepFade Exec 28
  • Set the stepfade to 50 % on executor 3.
[Channel]> StepFade 50 Exec 3
