Pause (large) Key

The large key    Pause    is located above the large key    Go -   .

 

  • Pressing    Pause    executes the DefGoPause keyword. The executors selected will be paused.
[time] : Macro : DefGoPause
  • Pressing    Pause    again executes the DefGoPause keyword again. The pause of the selected executors ends.
