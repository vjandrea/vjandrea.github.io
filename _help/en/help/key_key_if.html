If Key

The key If is located in the command area on the left of the key At.

 

If the command line is empty,

  • pressing If enters the IfOutput keyword 
[Channel]> IfOutput

 

  • pressing If If enters the IfActive keyword 
[Channel]> IfActive

 

  • pressing If If If enters the IfProg keyword 
[Channel]> IfProg

 

  • pressing If If If If enters the If keyword 
[Channel]> If

into the command line. 

 

If the command line contains previous input, the If key toggles.

  • Pressing If enters the If keyword into the command line. 
  • Pressing If again enters the EndIf keyword into the command line. 
[Channel]> Store If Group 5 EndIf
