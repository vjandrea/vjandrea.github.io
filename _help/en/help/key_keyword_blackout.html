Blackout keyword

Blackout is a function used to force zero values on output for dimmer parameter of channels and fixtures. Fixtures with "React to Master" = off in the fixture edit menu do not react to Blackout. As long as Blackout is enabled the "B.O." button is on i.e.,  it is permanently burning.

To execute this keyword press the key B.O., or type Blackout or the shortcut Bl into the command line. 

Syntax

Blackout

Blackout On

Blackout Off

 

Blackout is a toggle function. This means that entering Blackout without any helping keyword will toggle Blackout mode on/off.

 

Examples

 

[Channel]> Blackout On

Turns on Blackout mode.
