Preset Key

The key Preset is located in the command area on the right of the key Macro.


 

  • Pressing Preset enters the Preset keyword into the command line.
[Channel]> Preset
  • Pressing Preset Preset enters the Attribute keyword into the command line.
[Channel]> Attribute
  • Pressing Preset Preset Preset enters the Gel keyword into the command line.
[Channel]> Gel
  • Pressing and holding  + Preset enters the PresetType keyword into the command line.
[Channel]> PresetType
  • Pressing and holding List + Preset opens the temporary preset pool on screen 2 for the selected preset in the preset control bar.
