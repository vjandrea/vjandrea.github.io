ClearAll keyword

ClearAll is a function used to empty the entire programmer.

To execute this keyword type ClearAll or the shortcut ClearA into the command line. 

Syntax

ClearAll

The ClearAll function clears the selection and discards all values in the programmer.

Press the button Clear for slightly over two seconds to clear the selection and to discard all programmer values. 

For more information on how to use further functions of the keyword Clear see ClearActive and ClearSelection.
