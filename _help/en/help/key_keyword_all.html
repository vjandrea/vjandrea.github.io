All Keyword

To get the All keyword in the command line type All in the command line.

Description

All is a function used to set the MAtricks property Single X to none (0), which means to reselect all fixtures, (or reselect all columns of fixtures, if Interleave is active).

For more information see MAtricks.

Syntax

All

Example

  • To reselects all fixtures or columns type in the command line:
[Channel]> All
