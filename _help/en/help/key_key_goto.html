Goto Key

The key Goto is located in the command area on the right of the key Effect.

 

  • Pressing Goto enters the Goto keyword into the command line.
[Channel]> Goto
  • Pressing Goto Goto enters the Load keyword into the command line.
[Channel]> Load
  • Pressing and holding  + Goto enters the LoadNext keyword into the command line.
[Channel]> LoadNext
  • Pressing and holding  + Goto Goto enters the LoadPrev keyword into the command line.
[Channel]> LoadPrev
