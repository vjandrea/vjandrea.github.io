Lock Keyword

To get the Lock keyword in the command line, type Lock or the shortcut Loc in the command line.

Description

The lock keyword is a function keyword to lock objects.

A locked object is indicated by a yellow lock icon . A locked cue is indicated by a black background and a red font in the sequence executor sheet. A locked executor or sequence is indicated by a red executor number or red sequence number in the mini executor bar.

A locked object can not be changed or deleted.

Syntax

Lock [Object-list]

Lock Off [Object-list]

Example

  • To lock cue 3 of the selected executor, type in the command line:
[Channel]> Lock Cue 3
