EffectSec Keyword

To get the EffectSec keyword in the command line type EffectSec or EffectS in the command line.

Description

EffectSec is a helping keyword to indicate individual effect speed in seconds.

Syntax

EffectSec

EffectSec [Value-list]

Examples

  • To switches the encoders to adjust the EffectSpeed layer type in the command line:
[Channel]> EffectSec

 

  • To knocks out any effect speed values of the programmer for the selection/feature type in the command line:
[Channel]> Off EffectSec

 

  • To set the effect speed for the selection/feature to 2 seconds per cycle type in the command line:
[Channel]> EffectSec 2

 

  • To set the effect speed for "pan" for the selection to 4 seconds per cycle type in the command line:
[Channel]> Attribute "pan" At EffectSec 4
