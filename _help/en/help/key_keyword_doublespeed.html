DoubleSpeed Keyword

To get the DoubleSpeed keyword in the command line type DoubleSpeed or the shortcut DS in the command line.

Description

DoubleSpeed is a function to multiply the speed or the executor by 2.

Syntax

DoubleSpeed [Object-list]

Example

  • To double the speed of executor 5 type in the command line:
[Channel]> DoubleSpeed Executor 5
