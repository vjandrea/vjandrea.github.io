Solo Keyword

To get the Solo keyword in the command line, type Solo or the shortcut So in the command line.

If you press Solo, the command will be directly executed.

Description

The Solo keyword forces zero values for intensity output for attributes of all unselected fixtures or channels.
Only the selected fixtures or channels will have intensity output for attributes.

To remove a fixture from the solo function, go to the Setup I Patch & Fixture Schedule. In the column React to Master remove the On.

If Solo is used standalone, it toggles between Solo On and Solo Off.

If Solo is on and in the message center options  yes is selected for Solo the status, the solo icon is visible in the command line.

The Solo keyword is a function keyword.

Syntax

Solo

Solo On

Solo Off

Example

  • Turn Solo on.
[Channel]> Solo On
