Help Key

The key Help is located on the right of the key Backup.

 

  • Pressing Help enters the Help keyword into the command line.
[Channel]> Help
  • Pressing Help Help enters the CmdHelp keyword into the command line.
[Channel]> CmdHelp
  • Pressing and holding Help + pressing any other key on the console, for example Fixture, opens the help window to the key pressed.
