>>> [GoFastForward] Key

The key >>> [GoFastForward] is located in the command area on the right of the key Learn.

 

  • Pressing >>> enters >>> [GoFastForward] keyword into the command line.
[Channel]> >>>

 

  • Pressing >>> >>> enters Flash keyword into the command line.
[Channel]> Flash

 

  • Pressing >>> >>> >>> enters ToFull keyword into the command line.
[Channel]> ToFull
