ListVar Keyword

To get the ListVar keyword in the command line, type ListVar or the shortcut ListV in the command line.

Description

The ListVar keyword is a function keyword to get the variables and their values in the Command Line Feedback window displayed.

Syntax

ListVar [Filter]

Examples

  • To get all variables in the command line feedback window displayed, type in the command line:
[Channel]> ListVar

 

  • To get all variables beginning with f in the command line feedback window displayed, type in the command line:
[Channel]> ListVar f*
