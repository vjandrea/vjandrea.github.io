Connect and Power Up the Device

  1. Unpack the device.
  2. Remove the packaging and cushioning material. 
  3. Place the device indoors on a stable surface. 
  4. Download the grandMA2 onPC software on www.malighting.com
  5. Install the software on your PC or laptop. For minimum requirements see System Requirements grandMA2 in the grandMA2 user manual. 
  6. Connect the grandMA2 onPC command wing or the grandMA2 onPC fader wing with your PC or laptop using a USB cable.
  7. Start the grandMA2 console or grandMA2 onPC. 
  8. Switch on the grandMA2 onPC command wing or the grandMA2 onPC fader wing: 
    -Insert IEC connector into the corresponding jack. 
    -Connect the main power plug. 
    -Turn on the power switch:
    -Set it to position I. 
    -The power switch lights up in red.

To turn off the grandMA2 onPC command wing set the power switch to position O. 

Important:
Show data is only saved in the grandMA2 onPC software.
Important:
You can disconnect the grandMA2 onPC command wing using the grandMA2 onPC software. The DMX output will then be immediately interrupted.
