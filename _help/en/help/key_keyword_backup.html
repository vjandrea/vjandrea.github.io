Backup Keyword

To get the Backup keyword in the command line, type Backup or the shortcut Ba in the command line.

Description

The Backup keyword opens and close the Backup menu. For more information, see Using the Backup Menu.

Hint:
Another way to open and close the Backup menu is to press the Backup key.

Syntax

Backup

Example

To open the Backup menu type in the command line:

[Channel]>  Backup
