Version of grandMA desk or grandMA onPC (off-line)

You can determine by means of the version number of grandMA VPU if the software version fits to the software version of the desk or grandMA2 onPC.

grandMA VPU version number can be determined via:

  • Filename of the setup:
    MAVPU_v{VPU-Version}
  • Splash screen (appears while startup)
  • Info box (Menu - Help - Info)
  • VPU status bar

Example:

MAVPU_v3.0.0.n

 

3.0.0. GMA2-Streaming Version This number should correspond to the grandMA 2 streaming version
n   This number is for internal purpose
