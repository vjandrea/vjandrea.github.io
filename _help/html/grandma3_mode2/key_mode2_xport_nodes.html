grandMA3 xPort Nodes

Important:
We recommend formatting with every update.
Whenever formatting, the IP address will be reset back to default (DHCP). 
Important:
grandMA3 onPC 2Port Node 2k outputs 1,000 parameters when run in Mode2. 

It is possible to run grandMA3 nodes using Mode2. 

Requierement:

  • Download the grandMA3 software on www.malighting.com 
  • Extract the entire data from the zip.file and copy it onto a USB flash drive. 
  1. Insert the USB flash drive.
  2. Power up the grandMA3 xPort Node.
  3. Update notification appears.
  1.  Press the rotary knob.
  2. Formatting notification appears showing a progress bar. 
  1. Press the rotary knob before the progress bar completes.
  2. The grandMA3 xPort Node starts to update.
  1. Once the update has been completed, a notification appears. 
  1. Remove the USB flash drive.
  2. The display goes white.
  3. The grandMA3 xPort Node starts to run in Mode2.
