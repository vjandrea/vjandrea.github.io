Update Presets

It is possible to update the presets which you have altered. 

To do so:

  1. Tap the preset you would like to update once. 
  2. The fixtures are now selected. 
  3. Tap the same preset once again. 
  4. The fixtures are now actively in the programmer. 
  5. Change the value of this preset. 
  6. Then, press Update. 
  7. The pop-up Update opens. 
  1. To disable the filter, tap Preset Filter ON. The button toggles to Preset Filter OFF. 
  2. To choose the update option of the preset type, tap Selective. The button toggles between Selective, Global, Universal and Preset Type Default. 
  3. The button Original content toggles to Add new content. 
    -To update existing values by the new values, tap Original content.
    -To add additional data to the preset, along with existing values, tap Add new content. 
  4. If you have adjusted the settings to your needs, tap Update Preset.
  5. The preset has now been updated. 
Hint:
For more information on filter, selective, global, universal and default see Create presets.
