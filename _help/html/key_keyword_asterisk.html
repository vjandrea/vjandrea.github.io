Asterisk *

To get the Asterisk * in the command line press and hold Shift + 8.

Description

The Asterisk * is a wildcard character to substitute for any other character or characters in a name.

Examples

  • Select all fixtures stored in a group and a name beginning with "Mac".
 [Channel]> Group Mac*
  •  Select all fixtures with a name beginning with "backt" and ending with "blue".
 [Channel]> Fixture backt*blue
