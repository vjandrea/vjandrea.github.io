Alert keyword

Alert is a keyword which generates an alert message in the message center.

To execute the keyword Alert type Alert or the shortcut Ale into the command line. 

Syntax

Alert "text"

 

Typing the keyword Alert into the command line displays an alert message marked in red in the message center. 

 

For more information on the icons, the colors of messages, and the center itself see Message center. 

 

Example: 

 

[Channel]> Alert "coffee break"

 

The text coffee break is displayed in red in the message center.
