MAtricks Blocks

This topic illustrates how to use the Block X based on an example.

For information on the keyword see MAtricksBlocks. 

Requirement:

  • Fixtures are selected via the command line or keys.

For more information on how to select fixtures for the usage of MAtricks see MAtricks. 

1. Example

  • Apply Block X

Initial situation: 

  1. To apply Block X, tap the button on the left of the  sign displayed below Block X in the MAtricks toolbar. 

For more information on the toolbar see MAtricks toolbar. 

  1. The calculator opens.
    -Enter 3. 
  2. Press Please. 

-or-

  1. Use the command line as follows:
[Channel]> MAtricksBlocks 3
  1. Press Please.
  2. MAtricks button turns green in the MAtricks toolbar. 
  3. Press Next. 

The fixtures are selected in a block of 3 on the x-axis. 

  1. To jump to the next block, press Next. 

  1. To increase the number of fixtures in a block, press and hold  and Set and press Next.

  1. To decrease the number of fixtures in a block, press and hold  and Set and press Prev. 

  1. To disable MAtricks blocks, simultaneously press + Set + Prev + Next.

The MAtricksBlocks are disabled.
