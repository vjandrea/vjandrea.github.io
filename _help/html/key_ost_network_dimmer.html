Network Dimmer

Monitor and configure a dimMA system in the window MA Dimmer Network. 

Important:
This section deals with the configuration of MA NDPs in the view MA Dimmer Network. The configuration of MA NDPs in the window MA Network Configuration is dealt with in Control the MA NDP. 

Requirement:

  • Add MA NDPs to the network configuration

For more information see Adding devices to your session.

The dimMA Hierarchy

The dimMA hierarchy consists of racks, crates, modules, and dimmers. 

Rack:

A rack consists of 1 or 2 NDPs and at least 1 crate.

Crate:

A crate consists of 3 modules. 

Module:

A module consists of 1, 2 or 4 dimmers.

Dimmer: 

A dimmer holds all the corresponding configuration data.
