Command Section

The command section is the area used to enter complex commands into the command line using keys. 

This section is located on the right side of the console. 


Command section

 

To open the command section on the onPC press Command Overlay and the window Command Overlay opens. 

Important:
Single keys can have several functions depending on how often or how long you press and hold them.

 

For more information on the keys and their commands see Key overview.

 

 

1. Example:

  • Pressing Channel 1 At 50 enters the command into the command line.
  • Pressing Please consequently sets the dimmer in channel 1 to 50 percent.

 

2. Example:

  • Pressing Store Executor 5 Cue 3 enters the command into the command line. 
  • Pressing Please consequently stores cue 3 on the executor 5.
